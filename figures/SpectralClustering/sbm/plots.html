Consistent Spectral Clustering on Stochastic Block Model Datasets

We generated synthetic graphs using Stochastic Block Model (SBM) [Ref] to evaluate the performance of PCM for various graph sizes. The synthetic graph is generated by SBM with k-blocks. The nodes in each block are connected by an edge with probability 0.5, and the nodes in different blocks are connected with probability 0.1. In each time step, one edge is added to the graph according to the edge connection probability. We generated graphs with the number of nodes 50, 100, 500, and 1000.
